# /simulate — 텔레그램 없이 5문답 시연

> 사용 시점: 단계 4.5 (시연 5문답 검증). `/research-persona` 직후, `/setup-telegram` 전.

## 갱이가 할 일

### 1. 시스템 프롬프트 로딩

```bash
cat persona/system_prompt.md
```

### 2. 5 turn 시뮬레이션

`anthropic` Python SDK 로 직접 호출 (텔레그램 안 거침). Claude Code 가 *대화 시뮬레이터*가 되어 5번 묻고, *Claude API* 가 페르소나로 답한다.

```python
# 갱이가 미리 준비한 5문 (사용자가 바꿔도 OK)
SAMPLE_TURNS = [
    "안녕하세요. 자기소개 부탁드려요.",
    "오늘 날씨 어때요?",
    "당신이 가장 잘하는 게 뭐예요?",
    "고민이 있는데 들어 주실래요?",
    "감사합니다. 마지막 한 마디 부탁드려요.",
]
```

각 turn 응답을 갱이가 화면에 보여 주고, 사용자에게 묻는다.

### 3. 평가 체크리스트

각 turn 후 갱이가 자체 체크:
- [ ] 톤 키워드 3개 중 *2개 이상*이 응답에 살아 있나?
- [ ] 자기 호칭이 일관되나?
- [ ] 금기 1줄을 어기지 않았나?
- [ ] 응답 길이가 의도대로인가?

### 4. 사용자 결정

5 turn 끝나면:

```
갱이: 시연 끝났어요 🐾
  - tone 살아 있음: [O/X]
  - 자기 호칭 일관: [O/X]
  - 금기 위반: [없음/있음]

이대로 lock 하시겠어요? (Y/N)
  Y → step 5 (/setup-telegram) 으로 진행
  N → 어디를 고치고 싶으세요? (사용자 답에 따라 system_prompt.md 수정 후 다시 simulate)
```

## 금기

- 텔레그램 토큰 사용 금지 (이 단계는 *오프라인 시연*)
- 사용자 평가 없이 lock 하지 않기
