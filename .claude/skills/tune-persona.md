# /tune-persona — 페르소나 조정

> 사용 시점: 단계 8.5 (페르소나 조정). 본인 검증 (step 7) 또는 페어 검증 (step 8) 후.

## 갱이가 할 일

### 1. 응답 paste 받기

사용자가 텔레그램에서 받은 *이상한 응답* 1~3개를 paste 받는다.

```
사용자: "이거 받았는데, 갱이가 보기엔 어때?"

[paste된 응답 본문]
```

### 2. 진단

`persona/system_prompt.md` 와 paste 응답을 *나란히* 비교:

- 톤 키워드 3개 중 어긋난 게 있나?
- 자기 호칭이 깨졌나?
- 응답 길이가 의도와 다른가?
- 금기 1줄 위반했나?

### 3. 1~2줄 수정 제안

**큰 수정 금지** — 한 번에 1~2줄만 바꾼다. 큰 수정은 페르소나 정체성이 흔들려요.

```
갱이의 제안 🐾:

「응답 스타일」 섹션에 한 줄 추가:
  - 길이: 항상 3~5문장 이내, 추가 설명 요청 없이 마침

「금기」 섹션 강화:
  - 절대 "AI 어시스턴트입니다" 라고 자기소개 시작 금지
```

### 4. 적용은 사용자 동의 후

```bash
# 사용자가 OK 하면 edit
```

### 5. 재시작 안내

```
적용했어요. /bot stop && /bot start 으로 갱이가 다시 띄울게요 🐾
(코드 변경은 아니지만 .env / system_prompt.md 변경은 재시작 권장)
```

## 금기

- 한 번에 3줄 이상 고치지 않기 (정체성 흔들림)
- 사용자가 paste 한 응답을 *로그에 영구 저장* 하지 않기 (사적 대화)
- 페르소나를 *완전히 다른 캐릭터로 바꾸자* 고 제안하지 않기 (그건 W3 의 일)
